## 0152. TaskUpdate Adapter Implementation

### Background

The TaskUpdate tool is the workhorse of the new task system. It supports:

- Creating new tasks (by calling with a new task object)
- Updating existing task status, content, and metadata
- Deleting tasks

This adapter replaces the functionality previously handled through TodoWrite for modifications.

### Requirements

1. **TaskUpdate Tool Integration**: Properly call TaskUpdate with various operations
2. **Create Operation**: Support creating new tasks via TaskUpdate
3. **Update Operations**: Support updating:
   - Status (pending, in_progress, completed)
   - Content (description)
   - ActiveForm (current activity)
   - Dependencies
   - Metadata
4. **Delete Operation**: Support soft/hard delete via TaskUpdate
5. **Response Parsing**: Parse and return updated task information
6. **Error Handling**: Handle invalid IDs, tool unavailability, validation errors
