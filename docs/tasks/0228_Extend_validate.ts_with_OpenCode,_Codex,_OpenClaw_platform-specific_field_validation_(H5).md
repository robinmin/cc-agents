## 0228. Extend validate.ts with OpenCode, Codex, OpenClaw platform-specific field validation (H5)

### Background

validate.ts currently only validates Claude and Gemini platform-specific fields. Three platforms are missing field validation: OpenCode (hidden, steps, permissions), Codex (sandboxMode, reasoningEffort), and OpenClaw platform-specific fields. This means agents targeting these platforms pass validation without checking required fields.

### Requirements

H5: Add platform-specific field validation for OpenCode (hidden, steps, permissions), Codex (sandboxMode, reasoningEffort), and OpenClaw to validate.ts. Follow existing validation patterns used for Claude and Gemini. Ensure validation errors are descriptive and identify the missing/invalid fields. Add test coverage for the new platform validations.
