### Design


5-dimension evaluation system building on types.ts DimensionScore and MagentEvaluationReport. Weight profiles (standard/minimal/advanced) in evaluation.config.ts. Per-dimension scoring functions in evaluate.ts. Grade thresholds: A>=90, B>=80, C>=70, D>=60, F<60. Pass threshold: 75%.

### Solution


Build 5-dimension evaluation using the MagentEvaluationReport type from types.ts:
- evaluation.config.ts: Weight configs per profile, grade thresholds, dimension display names
- evaluate.ts: Per-dimension scoring functions (completeness checks section coverage, specificity checks for decision trees and examples, verifiability checks for anti-hallucination, safety checks for CRITICAL markers, evolution-readiness checks for memory/feedback).
