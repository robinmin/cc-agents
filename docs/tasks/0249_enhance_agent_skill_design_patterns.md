## 0249. enhance agent skill design patterns

### Background
When we develop the meta agent skill rd3:cc-skills, we already classified all agent skills into 3 category: technique, pattern, reference. We provide relevant templates to support it.

Recently, we noticed that Google cloud tech team provide a new and more mature category to classify their agent skills. You can refer to the document in @docs/reasearch/5_Agent_Skill_design_patterns_every_ADK_developer_should_know.md.

Furthermore, this documnet also provide a decetion tree for how to choice the right agent skill pattern, this also need to contained in the agent skill. it can help to make decition to find out the right pattern to work with.

### Requirements
Follow the new methodology to classify the agent skill to enhance existing meta agent skill rd3:cc-skills at plugins/rd3/skills/cc-skills.
