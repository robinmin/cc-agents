## 0230. Optimize adapt.ts to reuse parsed content instead of re-running pipeline per platform (H2)

### Background

In adapt.ts, the adaptToAll() function calls adaptAgent() for each platform, which re-runs the full parse+generate pipeline every time. This is inefficient when adapting to multiple platforms since the source agent content doesn't change between platforms. The parsed content should be computed once and reused across all platform adaptations.

### Requirements

H2: Refactor adaptToAll() in plugins/rd3/skills/cc-agents/scripts/adapt.ts to parse the source agent once, then pass the pre-parsed content to each platform adapter. Ensure adaptAgent() can accept pre-parsed content as an optional parameter. Maintain backward compatibility for single-platform adapt calls. Verify no behavioral change in output quality.
