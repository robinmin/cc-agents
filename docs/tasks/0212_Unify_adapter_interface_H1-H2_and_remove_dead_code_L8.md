## Batch 2: Adapter Interface Unification (H1-H2, L8)

Unify the adapter architecture so all 5 adapters share a common interface.

### H1: Adapters don't implement IPlatformAdapter
- `base.ts` defines `BaseAdapter` abstract class and `IPlatformAdapter` interface
- None of the 5 adapters (claude, codex, openclaw, opencode, antigravity) extend or implement them
- `BaseAdapter`, `AdapterRegistry`, `adapterRegistry`, `createAdapterContext` are dead code
- **Fix**: Either make adapters implement `IPlatformAdapter`/extend `BaseAdapter`, OR remove dead abstractions and create a lightweight shared interface that matches actual adapter shape
- **Files**: `scripts/adapters/base.ts`, `scripts/adapters/claude.ts`, `scripts/adapters/codex.ts`, `scripts/adapters/openclaw.ts`, `scripts/adapters/opencode.ts`, `scripts/adapters/antigravity.ts`

### H2: `any` type bypass in refine.ts
- Line 157: `platformAdapters: Record<string, any>` exists because adapters lack common interface
- **Fix**: Once H1 is resolved, replace `any` with the shared adapter interface type
- **Files**: `scripts/refine.ts`

### L8: Dead function _evaluatePlatform
- Line 903: `_evaluatePlatform` is dead code (related to C1, but if C1 wires it in, this becomes moot)
- **Fix**: Verify C1 addressed this; if function remains unused after C1, remove it
- **Files**: `scripts/evaluate.ts`

### Acceptance Criteria
- All adapters implement a shared interface
- No `any` types for adapter references
