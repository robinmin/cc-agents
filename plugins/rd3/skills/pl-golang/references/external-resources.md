# External Resources

Curated external resources for Go development.

## Official Documentation

- [Go Documentation](https://go.dev/doc/) - Official docs, tutorials, and references
- [Go Standard Library](https://pkg.go.dev/std) - Complete stdlib reference
- [Go Blog](https://go.dev/blog/) - In-depth articles from the Go team
- [Go Release Notes](https://go.dev/doc/devel/release) - Version-specific changes

## Learning Resources

### Getting Started
- [A Tour of Go](https://go.dev/tour/) - Interactive Go tutorial
- [Go by Example](https://gobyexample.com/) - Hands-on examples
- [Golang.org Tutorial](https://go.dev/doc/tutorial/getting-started) - Setup and first program

### Books
- [The Go Programming Language](https://www.gopl.io/) - Alan Donovan & Brian Kernighan
- [Go in Action](https://www.manning.com/books/go-in-action) - Williams, Ketelon, & Kessler
- [Programming in Go](https://www.amazon.com/Programming-Go-Wesley-Microsoft-Programmers/dp/0321774639)

## Best Practices

### Code Style
- [Effective Go](https://go.dev/doc/effective_go) - Official style guide
- [Go Code Review Comments](https://github.com/golang/go/wiki/CodeReviewComments) - Common patterns
- [Go Proverbs](https://go-proverbs.github.io/) - Rob Pike's design principles

### Project Structure
- [Standard Go Project Layout](https://github.com/golang-standards/project-layout) - Common directory structure
- [Structuring Go Applications](https://medium.com/golang-learn/structuring-go-applications-b5e6a0a140ea) - Architecture patterns

## Concurrency

- [Go Concurrency Patterns](https://go.dev/blog/pipelines) - Channels and pipelines
- [Share Memory by Communicating](https://go.dev/doc/effective_go#sharing) - Core philosophy
- [Advanced Go Concurrency](https://blog.golang.org/context) - Context patterns

## Testing

- [Testing in Go](https://go.dev/blog/testing) - Testing techniques
- [Fuzzing in Go](https://go.dev/security/fuzz/) - Property-based testing
- [Using Go's cover tool](https://blog.golang.org/cover) - Coverage analysis

## Tooling

- [Go Modules Reference](https://go.dev/ref/mod) - Dependency management
- [Go Workspaces](https://go.dev/blog/workspaces) - Monorepo support
- [go.dev/ref](https://go.dev/ref/) - Technical references

## Community

- [Go Forum](https://forum.golang.org/) - Discussion forum
- [Go Playground](https://go.dev/play/) - Run Go code in browser
- [r/golang](https://reddit.com/r/golang) - Reddit community

## Advanced Topics

- [Go Generics Draft](https://go.dev/blog/intro-generics) - Generics introduction
- [Error Handling in Go](https://go.dev/blog/error-handling) - Error patterns
- [Profiling Go Programs](https://go.dev/blog/profiling) - Performance optimization
