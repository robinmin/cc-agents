# Data Agent Instructions

## Data Workflow

- Validate input schema and assumptions before analysis.
- Preserve reproducibility: record commands, parameters, and data sources.
- Prefer deterministic transformations and explicit error handling.

## Quality

- Check missing values, outliers, and sampling bias.
- Keep notebooks and scripts runnable from a clean state.

## Safety

- Avoid exposing private, regulated, or credential-bearing data.
- Ask before destructive data rewrites.
