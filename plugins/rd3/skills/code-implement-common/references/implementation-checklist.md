# Implementation Checklist

Before marking implementation complete:

- [ ] Task file referenced in prompt
- [ ] Worktree created and isolated
- [ ] TDD cycle completed with fix-and-repeat until all tests pass
- [ ] Task file updated after each major step (progress persisted)
- [ ] Tests passing with 80%+ coverage
- [ ] Code follows project conventions
- [ ] Implementation matches requirements
- [ ] Worktree pushed to remote
- [ ] Review delegated (or --no-review used for batch)
- [ ] Worktree cleaned up (if --cleanup was used)
