# Style: Sketch

## Description

Hand-drawn, artistic feel with organic lines and pencil drawing aesthetic. Warm and approachable with natural imperfections.

## Visual Characteristics

- **Lines**: Organic, hand-drawn, varying stroke width
- **Texture**: Paper texture, pencil shading
- **Colors**: Warm tones, sepia, pencil grays
- **Style**: Artistic, informal, personal touch
- **Elements**: Sketchy details, natural imperfections

## Prompt Modifiers

```
hand-drawn sketch style, artistic, pencil drawing feel, organic lines, paper texture, warm tones, informal sketch, graphite pencil, natural imperfections, artistic rendering
```

## Use Cases

- Concept art
- Storyboards
- Informal diagrams
- Wireframe sketches
- Personal illustrations
- Educational visuals
