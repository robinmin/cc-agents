# Style: Minimalist

## Description

Simple, clean aesthetic with plenty of whitespace. Focus on essential elements with subtle colors and elegant composition.

## Visual Characteristics

- **Lines**: Thin, delicate, minimal strokes
- **Colors**: Subtle, muted palette (grays, soft blues, whites)
- **Layout**: Generous whitespace, balanced composition
- **Typography**: Clean sans-serif, generous spacing
- **Elements**: Simple shapes, negative space, minimal detail

## Prompt Modifiers

```
minimalist design, clean lines, simple aesthetic, plenty of whitespace, balanced composition, subtle colors, elegant, refined, modern minimalist, scandinavian design style
```

## Use Cases

- Modern UI mockups
- Simple illustrations
- Concept art
- Logo concepts
- Clean data visualizations
- Abstract representations
