# Style: Minimalist

## Description

Simple, clean aesthetic with plenty of whitespace. Focus on essential elements with subtle colors and elegant composition. Less is more — every element serves a purpose.

## Visual Characteristics

- **Colors**: Subtle, muted palette — soft grays, off-whites, one accent color
- **Color Palette**: `#FAFAFA` (near white), `#E0E0E0` (light gray), `#9E9E9E` (medium gray), `#424242` (dark gray), `#2196F3` (accent blue)
- **Lighting**: Even, soft, ambient — no dramatic shadows
- **Composition**: Generous whitespace, balanced, centered or rule-of-thirds, geometric alignment
- **Texture**: Smooth matte surfaces, clean edges, subtle paper grain
- **Typography**: Clean sans-serif, generous letter-spacing
- **Emotional Tone**: Calm, sophisticated, modern, intentional, refined
