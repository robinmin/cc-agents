# Style: Technical Diagram

## Description

Clean, technical illustrations with precise lines and professional diagram style. Minimal color palette focused on clarity and technical accuracy.

## Visual Characteristics

- **Lines**: Clean, precise, consistent stroke width
- **Colors**: Minimal, professional palette (blues, grays, white background)
- **Layout**: Organized, hierarchical, clear visual flow
- **Typography**: Sans-serif, technical, readable
- **Elements**: Boxes, arrows, connection lines, labels

## Prompt Modifiers

```
clean technical illustration, precise lines, professional diagram style, minimal colors, organized layout, clear visual hierarchy, technical drawing, blueprint style, architectural diagram
```

## Use Cases

- System architecture diagrams
- Flowcharts and process diagrams
- Network topology diagrams
- API interaction diagrams
- Database schema diagrams
- Component architecture diagrams

## Examples

**Input**: "Microservices architecture"
**Enhanced**: "Microservices architecture, clean technical illustration, precise lines, professional diagram style, minimal colors, organized layout, showing services, API gateway, load balancer, databases"

**Input**: "Data flow in ETL pipeline"
**Enhanced**: "ETL pipeline data flow diagram, clean technical illustration, precise lines, professional diagram style, showing extract, transform, load stages with data flow arrows"
