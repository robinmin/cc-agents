# Style: Technical Diagram

## Description

Clean, technical illustrations with precise lines and professional diagram style. Minimal color palette focused on clarity and technical accuracy. Communicates complexity with visual simplicity.

## Visual Characteristics

- **Colors**: Minimal, professional palette — blues, grays, white background, one highlight color
- **Color Palette**: `#FFFFFF` (white background), `#2196F3` (primary blue), `#90CAF9` (light blue), `#616161` (dark gray), `#FF9800` (highlight orange)
- **Lighting**: Flat, even — no dramatic shadows, consistent illumination
- **Composition**: Organized, hierarchical, clear visual flow, grid-aligned
- **Lines**: Clean, precise, consistent stroke width, orthogonal connections
- **Elements**: Boxes, arrows, connection lines, labels, numbered steps
- **Emotional Tone**: Authoritative, clear, educational, trustworthy, precise

## Prompt Modifiers

```
clean technical illustration, precise lines, professional diagram style, minimal colors, organized layout, clear visual hierarchy, technical drawing, blueprint style, architectural diagram, flat even lighting, grid-aligned composition
```

## Use Cases

- System architecture diagrams
- Flowcharts and process diagrams
- Network topology diagrams
- API interaction diagrams
- Database schema diagrams
- Component architecture diagrams

## Best Platform Pairings

- **Technical blogs**: Architecture documentation
- **GitHub README**: Project overviews
- **Documentation sites**: Clear technical communication
- **Slide decks**: Technical presentations
