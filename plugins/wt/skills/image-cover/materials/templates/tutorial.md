# Style Template: Tutorial

## Description

Instructional covers for how-to guides and tutorials with clear visual cues and step-by-step progression indicators.

## Visual Characteristics

- **Colors**: High contrast, educational, clear
- **Typography**: Instructional, easy to read, emphasis
- **Layout**: Step indicators, clear progression
- **Elements**: Numbers, arrows, progress indicators
- **Style**: Educational, clear, structured

## Prompt Template

```
{title}, {themes}, tutorial cover, step-by-step visual cues, instructional, clear progression, educational style, 2.35:1 cinematic aspect ratio, with title text overlay
```

## Style Modifiers

```
step-by-step visual cues, instructional, clear progression, educational style, high contrast, clear, structured, learning-focused
```

## Text Overlay

- **Font**: Clear, instructional sans-serif
- **Style**: Bold, educational, emphasis on learning
- **Position**: Clear visibility, numbered if applicable
- **Color**: High contrast for readability

## Color Palette

| Color | Usage |
|-------|-------|
| #27AE60 | Green (success, progress) |
| #F39C12 | Orange (steps, warnings) |
| #3498DB | Blue (information) |
| #2C3E50 | Dark (text) |

## Best For

- How-to guides
- Step-by-step tutorials
- Learning resources
- Educational content
- Instructional materials

## Example

**Article**: "Getting Started with Docker: A Complete Guide"

**Analysis**:
- Title: "Getting Started with Docker: A Complete Guide"
- Themes: Docker, getting started, containerization, guide
- Content Type: tutorial

**Enhanced Prompt**:
```
Getting Started with Docker, tutorial cover, step-by-step visual cues, instructional, clear progression, showing Docker container, terminal, step indicators, educational style, 2.35:1 cinematic aspect ratio, with title text overlay: "Getting Started with Docker: A Complete Guide"
```

## Detection Heuristics

Auto-detect tutorial style when article contains:
- Numbered steps or lists
- Tutorial-related terms (how-to, guide, getting started)
- Step indicators (Step 1, First, Next, Finally)
- Instructional language (learn, master, understand)
- Tutorial structure (introduction, prerequisites, steps)
