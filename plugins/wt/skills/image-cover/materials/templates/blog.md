# Style Template: Blog

## Description

Engaging, personal covers for blog posts with a warm, inviting aesthetic. Focus on readability and personal connection.

## Visual Characteristics

- **Colors**: Warm, inviting, brand-aligned
- **Typography**: Friendly, readable, personal
- **Layout**: Balanced, accessible, engaging
- **Elements**: Personal touches, imagery, illustrations
- **Style**: Modern, approachable, conversational

## Prompt Template

```
{title}, {themes}, blog post cover, engaging, readable, personal touch, modern blog aesthetic, 2.35:1 cinematic aspect ratio, with title text overlay
```

## Style Modifiers

```
engaging, readable, personal touch, modern blog aesthetic, warm, inviting, approachable, conversational, friendly
```

## Text Overlay

- **Font**: Friendly sans-serif (Open Sans, Lato)
- **Style**: Approachable, mixed case, bold
- **Position**: Center with visual interest
- **Color**: White with shadow or dark with light background

## Color Palette

| Color | Usage |
|-------|-------|
| #FF6B6B | Warm accent |
| #4ECDC4 | Teal accent |
| #FFE66D | Yellow highlight |
| #2C3E50 | Dark text |

## Best For

- Personal blog posts
- Opinion pieces
- Stories and narratives
- Thought leadership
- Personal experiences

## Example

**Article**: "My Journey Learning React"

**Analysis**:
- Title: "My Journey Learning React"
- Themes: learning, React, journey, web development
- Content Type: blog

**Enhanced Prompt**:
```
My Journey Learning React, blog post cover, engaging, readable, personal touch, modern blog aesthetic, showing learning journey, code editor, progress visualization, warm colors, 2.35:1 cinematic aspect ratio, with title text overlay: "My Journey Learning React"
```

## Detection Heuristics

Auto-detect blog style when article contains:
- Personal pronouns (I, my, me)
- Conversational tone
- Story-telling structure
- First-person narrative
- Opinion or experience sharing
