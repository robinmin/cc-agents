# Style Template: News

## Description

Professional, headline-focused covers for news articles and announcements with journalistic aesthetic and formal presentation.

## Visual Characteristics

- **Colors**: Professional, newsroom aesthetic
- **Typography**: Headline style, journalistic, formal
- **Layout**: Headline-focused, balanced, structured
- **Elements**: Datelines, formal indicators
- **Style**: Professional, journalistic, credible

## Prompt Template

```
{title}, {themes}, news article cover, professional, headline-focused, journalistic style, newsroom aesthetic, 2.35:1 cinematic aspect ratio, with title text overlay
```

## Style Modifiers

```
professional, headline-focused, journalistic style, newsroom aesthetic, credible, formal, balanced, structured
```

## Text Overlay

- **Font**: Headline serif or formal sans-serif
- **Style**: Headline style, impactful, journalistic
- **Position**: Prominent, headline placement
- **Color**: High contrast, professional

## Color Palette

| Color | Usage |
|-------|-------|
| #C0392B | Red (breaking, urgent) |
| #2C3E50 | Navy (formal, credible) |
| #7F8C8D | Gray (supporting) |
| #ECF0F1 | Light (background) |

## Best For

- News articles
- Announcements
- Press releases
- Industry updates
- Breaking news

## Example

**Article**: "React 19 Released: Major Performance Improvements"

**Analysis**:
- Title: "React 19 Released: Major Performance Improvements"
- Themes: React 19, release, performance, update
- Content Type: news

**Enhanced Prompt**:
```
React 19 Released: Major Performance Improvements, news article cover, professional, headline-focused, journalistic style, showing React logo, performance indicators, update visuals, newsroom aesthetic, 2.35:1 cinematic aspect ratio, with title text overlay: "React 19 Released: Major Performance Improvements"
```

## Detection Heuristics

Auto-detect news style when article contains:
- Datelines or timestamps
- Announcement language (released, announced, launched)
- Formal, objective tone
- News structure (headline, dateline, body)
- Industry or company updates
- Breaking or urgent indicators
