# AI Coding Best Practices Thread

1/ AI coding assistants are powerful tools. Here are best practices to use them effectively:

- Understand code before accepting
- Apply same review standards as human code
- Protect sensitive information

2/ Security matters:

- Never share API keys or secrets
- Be aware of data privacy
- Scan for hardcoded secrets

3/ Use AI as a learning tool:

- Ask for explanations
- Build mental models
- Do not skip understanding

4/ Key takeaways:

- Review rigorously
- Protect sensitive data
- Learn continuously
