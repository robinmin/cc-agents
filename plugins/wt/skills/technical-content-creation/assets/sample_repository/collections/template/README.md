# Collection Template

This folder contains the template structure for new collections.

## Contents

- `.topic.md` - Collection metadata
- `README.md` - Collection documentation
- `topics/` - Folder for individual topics

## Usage

Copy this template folder when creating a new collection, then:
1. Rename the folder to your collection name
2. Update `.topic.md` with collection details
3. Update `README.md` with collection description
4. Add topics to the `topics/` folder

## Metadata Fields

See `.topic.md` for required and optional fields.
