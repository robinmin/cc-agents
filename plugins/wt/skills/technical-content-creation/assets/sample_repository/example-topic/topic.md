## Additional Notes

This is an example topic that demonstrates the complete folder structure.

## How to Use

1. Copy this entire folder to create a new topic
2. Update `topic.md` with your topic information
3. Replace placeholder content with your actual content
4. Follow the stage progression (0-6)

## Stages

- **0-materials/**: Collect and organize research sources
- **1-research/**: Analyze and synthesize findings
- **2-outline/**: Create content structure
- **3-draft/**: Write initial content
- **4-illustration/**: Create visual aids
- **5-adaptation/**: Adapt for different platforms
- **6-publish/**: Finalize and publish content
